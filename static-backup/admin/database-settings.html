Why Supabase?

Get Started at Supabase.com

Quick Setup (5 minutes)

  1. Go to supabase.com and create a free account
  2. Create a new project (choose any region)
  3. Wait 2 minutes for project to be ready
  4. Go to Settings → API in your Supabase dashboard
  5. Copy the Project URL and anon public key below
  6. Click "Create Tables" below to set up your database
